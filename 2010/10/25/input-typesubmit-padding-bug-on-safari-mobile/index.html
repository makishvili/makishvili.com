Неуправляемый паддинг внутри Input

Понедельник, 25 Октября 2010

В мобильном Сафари внутри input появляется неуправляемый паддинг слева и справа от текста. Прочитать подробное описание проблемы.

Решение — использовать button вместо input:

<style>
    button {font-size: 1px}
    button span {font-size: 14px}
</style>
<button><span>Текст кнопки</span></button>

Решение найдено методом “мгновенного коллективного озарения” :) Озарило Антона, Наташу, Олю
